Detailed Description

spiReader ADT class

A ADT class for a reader that implements SPI, built using hwlib./n You need to supply a hwlib pin_out for the nss pin,/n you also has to supply a hwlib spi_bus_banged_sclk_mosi_miso bus for the spiBus./n You also has to define the read and write funcions when you inherit from this class.
